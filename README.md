# Invoice Management Script

Paylance is a modern and user-friendly Invoice Management Script designed for freelancers, individuals, and small to medium-sized businesses who want full control over their billing system. As a self-hosted solution, it gives you complete ownership of your data while providing all essential tools to streamline invoicing and payment management. Built using the robust CodeIgniter PHP framework and powered by a MySQL database, Paylance ensures performance, scalability, and ease of customization. The system is thoughtfully designed with a clean interface, making it easy for both technical and non-technical users to operate.

With Paylance, you can efficiently manage your clients, create and track invoices, record payments, and generate professional PDF invoices—all from a centralized dashboard. Whether you're running a freelance business or managing multiple clients, this script simplifies your workflow and saves valuable time.
